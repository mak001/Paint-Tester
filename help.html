Graphics Frame

Shows a graphical representation of the last saved state of the project.

Code Frame (General)

The Code Frame is where everything except for the graphical rendering happens. The next few sections will cover the Code Frame with more detail.

File Menu

New - creates a new project, just as if you started the paint tester.

Save - saves the current project. If the p[roject was not previously saved this option would act as Save As does. Updates the Graphics Frame.

Save As... - saves a project, asking where to save it and what to name it. Updates the Graphics Frame.

Open - opens a pre-existing project. Updates the Graphics Frame.

Exit - exits the program, just like if you hit the "X" button on the Code or Graphics Frame.

Edit Menu

Import - imports a pre-existing .java file. (does NOT work in v0.3 or lower).

Open a closed file - opens a closed tab. (One that had the "X" clicked).

Close All - Closes all the opened tabs. (tabs can be reopened anytime using the "Open a closed file" option).

Settings - Lets you take control of how the program handles itself with the current project.

Manipulate variables - Allows for manipultaion of skill xps and skill levels.

Reload project - Loads the project from the previous saved state.

Help Menu

Help - Shows a dialogue with the information on this page.

About - Shows a dialogue with a short version of the about page.

Site - Opens the default web browser to the project's home page.

Credits - Shows a list of credits.
