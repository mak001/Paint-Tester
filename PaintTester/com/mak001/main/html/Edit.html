Import

Have a class that is already written but don't want to copy it, or have a class that you wish to share between projects? Use this option to navigate to the class to make it part of the current project.

Open a closed file

Closed a tab and need to open it? use this option to choose which tab(s) to open. (hold ctrl to select multiple tabs to open)

Close all

Simply closes all the open tabs.

Settings

This will allow you to change how the PaintTester works. Here are an explanation of what each option does:

  • Copy imported files - If selected, this option will copy any file you decide to import into the src directory of the current project.
  • Use src folder - if selected, this option will make a folder named "src" in the directory of the current project's save. The src folder will be used to store the source files.
  • Use bin folder - if selected, this option will make a folder named "bin" in the directory of the current project's save. The bin folder will be used to store the compiled files.
  • Bot API to use - this will make it auto generate the first tab opened acording to the current bot selected. (would do extends ActiveScript for PowerBot, or extends Script for RuneDream)
Manipulate variables

This option will allow you to manipulate very easily. The major ticks on the levels go by 9. The major ticks on the experience go by 1,000,000.

Reload

This will reload the current project. It will load the last save. (Use only if you can do with loosing work)
