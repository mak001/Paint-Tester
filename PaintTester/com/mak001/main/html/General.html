What is this?

This is a program made by mak001 to help design paints for Powerbot scripts. Although it is far from a replica, it is based off of Enfilade's Easel. You can code and test a paint with this program, although it is advised to use an official IDE to code the paint.

Not sure where to start?

The button in the code frame will add a new tab that can be used to write a class. The first time the button is pressed it will generate code to look like a skeleton of a script. There can only be one script skeleton per project.

After clicking the plus button a pop up will appear asking what you want to name your class. It can be anything, but things that are predefined in Java like modifiers and primitives although classes are an exception(no classes named int, but you could name one ArrayList).

The tab that appears will have the name of the class written on it and will have a . This will remove the tab from the code frame, but will not completely get rid of the tab. See Edit -> Open a closed file for more information.

You could also click one of the options on the left side to read other parts of the help file.
